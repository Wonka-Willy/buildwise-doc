## Known Issues & Fixes

### Authentication & Access

<Tabs>
<Tab title="Login Issues">
**Issue:** Rate limiting triggers too quickly  
**Details:** "Too many attempts" appears after fewer than 5 clicks  
**Status:** Being optimized  
**Workaround:** Wait 15 minutes between attempts

**Issue:** Verification links expire quickly  
**Details:** Email verification links valid for only a few minutes  
**Status:** Extending validity period  
**Workaround:** Request new verification immediately if link fails
</Tab>

<Tab title="Email Problems">
**Issue:** Verification emails in spam  
**Details:** System emails often end up in junk folders  
**Status:** Improving email deliverability  
**Workaround:** Check spam folder, whitelist @tooli.be domain

**Issue:** Dark mode email visibility  
**Details:** Some email buttons not visible in dark mode  
**Status:** Fixing email templates  
**Workaround:** Switch to light mode for emails
</Tab>

<Tab title="Account Management">
**Issue:** Password change not working  
**Details:** Change password function disabled  
**Status:** Security review in progress  
**Workaround:** Contact support for password reset

**Issue:** Account deletion incomplete  
**Details:** Users can log back in after account deletion  
**Status:** Fixing data purge process  
**Workaround:** Contact support for complete account removal
</Tab>
</Tabs>

## Reporting Issues

### How to Report
<Steps>
<Step title="Check This Page">
Verify the issue isn't already listed in known issues
</Step>

<Step title="Gather Information">
Screenshot error messages, note reproduction steps, include browser/device info
</Step>

<Step title="Contact Support">
Email support@tooli.be with detailed information
</Step>

<Step title="Follow Up">
You'll receive a ticket number to track resolution progress
</Step>
</Steps>

### What We Need
- **Specific steps** to reproduce the issue
- **Error messages** (exact text or screenshots)  
- **Browser and device** information
- **Time and date** when issue occurred
- **Your organization** and account email
